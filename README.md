# Blood-Bank-Management-System
A web-based Blood Bank Management System built using Django that streamlines donor registration, blood inventory management, blood requests, and administrative operations, ensuring efficient and reliable blood donation services.

## Features
- Donor Registration and Management
- Blood Group Search
- Blood Request Management
- Admin Dashboard
- User-Friendly Interface
- Secure Data Management

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite Database
